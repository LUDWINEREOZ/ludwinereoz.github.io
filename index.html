Conoce a Nereo

¡Hola! Soy Nereo, un apasionado desarrollador de aplicaciones web con un enfoque vanguardista. Mi fascinación por la tecnología me ha llevado a explorar las infinitas posibilidades de integrar el mundo digital con el físico, especialmente en el ámbito del Internet de las Cosas y la Industria 4.0. Mi objetivo es crear soluciones innovadoras que no solo optimicen la experiencia web, sino que también impulsen la eficiencia y conectividad en la era digital. Bienvenido a mi espacio, donde la creatividad y la tecnología se encuentran para transformar ideas en realidades digitales emocionantes.

Fronted

Como desarrollador frontend, domino HTML, CSS y JavaScript, y tengo experiencia destacada en la construcción eficiente de interfaces web atractivas utilizando React. Mi enfoque se centra en la creación de experiencias intuitivas y agradables, respaldado por habilidades sólidas en diseño responsivo y optimización del rendimiento.

Backend

Como desarrollador backend, cuento con una sólida competencia en tecnologías clave como Spring, Express, .NET y C#. Mi experiencia abarca la implementación eficiente de lógica de servidor, creación de API robustas y gestión efectiva de bases de datos para garantizar un rendimiento óptimo. He trabajado en proyectos que requieren un manejo eficiente de datos y una arquitectura backend sólida.

Spring Express .Net C#

Base de Datos y Nube
